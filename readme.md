# Scripts

## Scope

Grab These Features in Markdown

- Single Line Comments
- All Headers
- Lists
- comments

And Render them into HTML


## Header Files

The Markdown Text

```md
# My Header
```

should render this

```html
<h1>My Header</h1>
<br/>
```

but this markdown text

```md
#MyHeader
```

should render this html

```html
<p>#MyHeader</p>
```
